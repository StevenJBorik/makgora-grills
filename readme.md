## DESIGN PHILOSOPHY (IMPORTANT – READ FIRST)

These grills are **not oversized novelty tusks**. The goal is *"feels like nothing"* wear:

* No forward weight beyond the natural tooth profile
* Visual tusk presence created through **contour illusion**, not protrusion
* All designs hug the tooth envelope or wrap the **base** of a tooth rather than extending outward
* Any apparent tusk length is achieved by **tapered geometry, shading, and color contrast**, not mass

> **Key principle:** The eye sees tusk. The mouth feels tooth.

## DENTAL SCAN REQUIREMENTS

### Accepted Inputs
- **Digital scan:** STL, PLY, or OBJ from intraoral scanner
- **Physical impression:** Mailed PVS impression kit (we digitize)

### File Specs
- Resolution: 50 micron minimum
- Mesh: 50,000+ triangles per arch
- Scope: Target teeth + 2mm gumline minimum
- Quality: No holes, no artifacts

### Compatible Scanners
iTero Element, 3Shape TRIOS, Carestream CS 3600, Primescan

> See `/research/dental-scans.md` for full requirements
> See `/scans/scan-requirements.md` for customer-facing docs

## STRUCTURAL COMFORT SYSTEM (GLOBAL)

* **Profile thickness:** 0.8–1.4mm visible, 0.6mm inner shell
* **Zero-pressure gum zones** flagged in CAD
* **Weight bias:** lingual-side hollowing
* **Retention:** friction-fit + micro-undercut (no snaps)
* **Optional soft liner:** biocompatible resin skin

✔ Protruding-tooth *illusion* is possible
✔ True protruding teeth are NOT recommended for daily wear
✔ Asymmetrical silhouettes create aggression without discomfort
