## Prompt Structure

Every prompt follows this structure:

```
1. CONTEXT BLOCK (always included)
2. ARCHETYPE + PALETTE + MOOD
3. ARCHETYPE-SPECIFIC DETAILS
4. GEOMETRY CONSTRAINTS
5. MATERIAL / COLOR DETAILS
6. DETAIL FEATURES
7. COMFORT REQUIREMENTS
8. OUTPUT SPECIFICATION
```

---

## Usage Tips

1. **Copy the full prompt** into Zoo CAD
2. **Modify the GEOMETRY section** if you have a specific dental scan
3. **Adjust MOOD** for different vibes (Warchief = aggressive, Royal = refined, etc.)
4. **Check OUTPUT** - request separate STLs for metal inlays
