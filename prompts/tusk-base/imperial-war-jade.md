- Primary: Base #0F5132, Highlight #2FA36B, Vein #B9E5C9

- Vein channels: Mark 0.3mm grooves for vein color application
- Gold inlay: Separate body, 0.4mm wide band

COMFORT REQUIREMENTS:
- Zero gum pressure (0.5mm clearance at gumline)

- Lingual hollowing for weight reduction
- Retention: friction-fit via 0.25mm interproximal undercuts

OUTPUT SPECIFICATION:
